# 📂 Референца за JS Компоненти: Индекс (Components Reference Index)

Овој индекс ги специфицира сите клучни интерактивни и податочни компоненти во `ln-ashlar`. Секоја компонента е детално документирана во сопствен документ во овој директориум, со својот селектор, детален HTML маркап и евенти.

---

## 1. Интерактивни Примитиви и Едноставни Компоненти

Ова се изолирани компоненти кои имаат за цел да управуваат само со својот дел од DOM дрвото и сопствените пристапни (ARIA) својства, целосно несвесни за околните компоненти.

* **[ln-toggle](components/ln-toggle.md)** — 🟢 Едноставна компонента
  * *Управува со отворање/затворање на панели, странични менија и слични транзиции.*
* **[ln-modal](components/ln-modal.md)** — 🟢 Едноставна компонента
  * *Управува со модални прозорци, заробување фокус и спречување скролање на страницата.*
* **[ln-popover](components/ln-popover.md)** — 🟢 Едноставна компонента
  * *Комплексни лебдечки контејнери за информации со автоматско прилагодување на висината/ориентацијата.*
* **[ln-toast](components/ln-toast.md)** — 🟢 Едноставна компонента / Сервис
  * *Слуша глобални window настани и динамички прикажува инстант известувања.*
* **[ln-table](components/ln-table.md)** — 🟢 Едноставна компонента / Презентер
  * *Stateless компонента за цртање табели на база на релативни настани за измена во кешот.*
* **[ln-search](components/ln-search.md)** — 🟢 Едноставна компонента
  * *Пресретнува текст во полето за пребарување, го дебаунсира внесот и управува со автоматско филтрирање на елементи.*
* **[ln-form](components/ln-form.md)** — 🟢 Едноставна компонента
  * *Врши брзо пополнување (fill) на полиња од рекорди и пренапишува REST рути.*
* **[ln-validate](components/ln-validate.md)** — 🟢 Едноставна компонента
  * *Се врзува за ValidityState на прелистувачот и визуелно ги прикажува/крие пораките за грешка на полињата.*
* **[ln-data-store](components/ln-data-store.md)** — 🟢 Едноставна компонента / Локална База
  * *Чист IndexedDB локален кеш кој нема познавања од мрежа и API патеки.*

---

## 2. Координатори

Ова се компоненти кои немаат сопствени посебни визуелни стилови, туку нивната одговорност е да слушаат настани од DOM-от и да координираат/менуваат други примитиви во системот.

* **[ln-accordion](components/ln-accordion.md)** — ⚙️ Координатор
  * *Ги слуша `ln-toggle:open` настаните и ги затвора сите останати отворени панели во истата група преку `setAttribute`.*
* **[ln-dropdown](components/ln-dropdown.md)** — ⚙️ Координатор
  * *Го координира активаторот со неговото `ln-toggle` мени, овозможува промоција на менито во Top Layer преку Popover API и динамички го позиционира на екран.*
* **[ln-data-coordinator](components/ln-data-coordinator.md)** — ⚙️ Координатор
  * *Централниот координатор на податоци кој ги поврзува локалниот кеш (`ln-data-store`), мрежниот приклучок (`ln-*-connector`) и офлајн редицата (`ln-api-queue`).*

---

## 3. Системски Помошници (System Utilities)

Ова се јадрени сервисни модули кои помагаат во олеснување на заедничките однесувања како што е зачувувањето на состојбите.

* **[ln-persist](components/ln-persist.md)** — ⚙️ Системски помошник / Модул за перзистенција (Persistence Utility Module)
  * *Служи за лесно зачувување и вчитување на корисничката состојба во `localStorage` за повеќе компоненти.*
* **[ln-fill](components/ln-fill.md)** — ⚙️ Системски помошник / Декларативен Активатор на Пополнување (System Utility / Declarative Fill Trigger)
  * *Служи за декларативно пополнување на форми преку клик на тригери кои содржат `data-ln-fill-*` атрибути.*
