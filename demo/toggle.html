Toggle

Generic toggle component — adds/removes the .open class on an element. Works for sidebar, collapsible, dropdown — anything.

Basics

AttributeOn elementDescription
data-ln-toggletarget elementCreates an instance, starts closed
data-ln-toggle="open"target elementStarts open (receives the .open class)
data-ln-toggle-for="id"buttonReferences the target element by ID
data-ln-toggle-action="open|close"buttonExplicit action (default: toggle)

Instance API

var el = document.getElementById('my-element');

el.lnToggle.open();      // Open
el.lnToggle.close();     // Close
el.lnToggle.toggle();    // Toggle
el.lnToggle.isOpen;      // boolean (read-only)

Events

EventBubblesDetail
ln-toggle:openyes{ target: HTMLElement }
ln-toggle:closeyes{ target: HTMLElement }
 
document.addEventListener('ln-toggle:open', function (e) {
    console.log('Opened:', e.detail.target.id);
});

document.addEventListener('ln-toggle:close', function (e) {
    console.log('Closed:', e.detail.target.id);
});

HTML structure

<!-- Element that will be toggled -->
<aside id="my-sidebar" data-ln-toggle>
    Content
</aside>

<!-- Button that controls it -->
<button data-ln-toggle-for="my-sidebar">
    Toggle
</button>

<!-- Starts open -->
<section id="panel" data-ln-toggle="open">
    Open content
</section>

<!-- Explicit action -->
<button data-ln-toggle-for="panel"
        data-ln-toggle-action="close">
    Close
</button>

CSS for animations

ln-toggle adds/removes the .open class. For CSS animations use the collapsible pattern:

// Sidebar — transform animation
.sidebar {
    transform: translateX(-100%);
    @include transition;
    &.open { transform: translateX(0); }
}

// Collapsible — grid-template-rows (RECOMMENDED)
// Use .collapsible class or @include collapsible
<section id="panel" data-ln-toggle class="collapsible">
    <article class="collapsible-body">
        Content...
    </article>
</section>

// SCSS:
.collapsible      { @include collapsible; }
.collapsible > *  { @include collapsible-content; }

// Or semantically in project SCSS:
#panel         { @include collapsible; }
#panel > .body { @include collapsible-content; }

Dynamic DOM

The component uses MutationObserver — it automatically initializes all new elements added to the DOM, including dynamic HTML from AJAX or JS.

// No manual initialization needed.
// MutationObserver automatically detects new data-ln-toggle elements.

fetch('/partial').then(r => r.text()).then(html => {
    document.getElementById('content').innerHTML = html;
    // lnToggle automatically initializes the new elements
});

window.lnToggle is available only for non-standard cases (Shadow DOM, iframe) where MutationObserver does not observe.

Practical uses

  • Sidebar navigation — opening/closing on mobile
  • Collapsible FAQ/documentation — Q&A panels
  • Dropdown menus — user menu, actions menu
  • Modal backdrop — via ln-modal component
  • Tabs panels — via ln-tabs component
  • Accordion — via ln-accordion component
