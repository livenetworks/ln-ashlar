    Dropdown

    A wrapper over ln-toggle that adds outside-click closing. Delegates all state to ln-toggle.

    Basics

    AttributeOn elementDescription
    data-ln-dropdownwrapperCreates a dropdown instance
    data-ln-togglemenuThe toggle target (menu element)
    data-ln-toggle-forbuttonTrigger button, references menu by ID

    Example — Basic Dropdown

    Example — Multiple Dropdowns

    Each dropdown operates independently. Opening one does not close the other (unless you click outside).

    API & Events

    Constructor

    // Manual initialization
window.lnDropdown(document.body);

    Events

    EventBubblesDetail
    ln-dropdown:openyes{ target }
    ln-dropdown:closeyes{ target }
    ln-dropdown:destroyedyes{ target }

    Open/close state is managed by ln-toggle. Use ln-toggle:before-open / ln-toggle:before-close to cancel.

    HTML Structure

    <div data-ln-dropdown>
    <button data-ln-toggle-for="my-menu">Open</button>
    <ul id="my-menu" data-ln-toggle>
        <li><button>Option A</button></li>
        <li><button>Option B</button></li>
    </ul>
</div>

    Dependencies

    Dropdown depends on ln-toggle. ln-toggle must be loaded before dropdown.

    <script src="dist/ln-acme.iife.js"></script>
