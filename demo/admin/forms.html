Forms & Buttons

CSS Grid + wrapping <label> pattern. No <div class="form-group"> or <div class="form-row">. The grid controls the layout via SCSS — not via HTML.

Principle

Each field is a <p class="form-element"> containing a <label for="..."> and <input id="..."> (explicit association). The root <form> gets @include form-grid in SCSS — a 6-column grid, collapsing to 1 column on mobile. Column widths are set via nth-child in form-specific SCSS.

Wrong — wrapping label

<!-- implicit label, no for/id -->
<label>
  Name
  <input type="text" name="name">
</label>

<label>
  City
  <input type="text" name="city">
</label>

Correct — form-element

<!-- explicit for/id, p wrapper -->
<form id="my-form">

  <p class="form-element">
    <label for="name">Name</label>
    <input type="text" id="name" name="name">
  </p>

  <p class="form-element">
    <label for="city">City</label>
    <input type="text" id="city" name="city">
  </p>

</form>
Element Rule
Root <form id="..."> — styled with @include form-grid in SCSS (6 cols → 1 col on mobile)
Children <p class="form-element"> — wraps <label> + <input>
Label Explicit for/id association: <label for="name"> + <input id="name">
Column spans In form-specific SCSS via .form-element:nth-child(N) — NEVER width classes on the element
Errors <small class="text-error"> inside .form-element
Pill radio/checkbox <ul> > <li> > <label> > <input> — grouped pills (see Pill Labels section)
.form-actions Component class — stays in HTML. Gets grid-column: span 6 (full-width) in SCSS

Buttons

Every <button> gets hover/active/focus effects out of the box from ln-acme via --color-primary. Override --color-primary on the element or parent to change color.

Implementation

HTML

<form id="my-form">

  <p class="form-element">
    <label for="name">Name <span class="text-error">*</span></label>
    <input type="text" id="name" name="name" required>
    <small class="text-error">Required field</small>
  </p>

  <p class="form-element">
    <label for="category">Category</label>
    <select id="category" name="category">
      <option value="a">Option A</option>
    </select>
  </p>

  <p class="form-element">
    <label for="notes">Notes</label>
    <textarea id="notes" name="notes"></textarea>
  </p>

  <div class="form-actions">
    <button type="button">Cancel</button>
    <button type="submit">Save</button>
  </div>

</form>

SCSS

#my-form {
  @include form-grid;

  .form-element { grid-column: span 3; }                   // default: half
  .form-element:nth-child(5) { grid-column: span 6; }     // notes: full
  > fieldset { grid-column: span 6; }
  .form-actions { grid-column: span 6; }
}
