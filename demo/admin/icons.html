    Icons

    On-demand SVG icon loader — ln-icons.js scans the DOM for <use href="#ln-*">, fetches each icon from Tabler Icons via jsDelivr CDN, and builds a hidden sprite at runtime. Any Tabler icon name works. No bundled SVG content, no init call needed.

    How it works

    Icons are fetched on demand from the Tabler Icons CDN (jsDelivr) and assembled into a hidden sprite in <body>. Any icon from Tabler works — just use its name as #ln-{name}. MutationObserver picks up icons added after page load automatically.

    Exception: #lnc-file-pdf, #lnc-file-doc, #lnc-file-epub are custom multi-color icons served from your own CDN — they do not follow currentColor.

    <!-- Standalone icon -->
<svg class="ln-icon" aria-hidden="true"><use href="#ln-home"></use></svg>

<!-- Icon in a button with text -->
<button>
    <svg class="ln-icon" aria-hidden="true"><use href="#ln-plus"></use></svg>
    Add User
</button>

<!-- Icon-only button — aria-label required -->
<button aria-label="Close">
    <svg class="ln-icon" aria-hidden="true"><use href="#ln-x"></use></svg>
</button>

<!-- Accordion chevron — CSS rotates on open -->
<header data-ln-toggle-for="panel1">
    Section Title
    <svg class="ln-icon ln-chevron" aria-hidden="true"><use href="#ln-arrow-down"></use></svg>
</header>

<!-- Custom icon (requires window.LN_ICONS_CUSTOM_CDN) -->
<svg class="ln-icon" aria-hidden="true"><use href="#lnc-file-pdf"></use></svg>

    Custom Icons (lnc-)

    Custom icons use the #lnc- prefix and are served from your own CDN. Set window.LN_ICONS_CUSTOM_CDN before the script loads:

    <script>
    window.LN_ICONS_CUSTOM_CDN = 'https://your-cdn.com/icons';
</script>
<script src="dist/ln-acme.iife.js" defer></script>

<!-- Use custom icons the same way -->
<svg class="ln-icon" aria-hidden="true"><use href="#lnc-my-icon"></use></svg>

    To add a new custom icon: create js/ln-icons/icons/{name}.svg with stroke="currentColor", run npm run build, then upload dist/icons/{name}.svg to your CDN.
