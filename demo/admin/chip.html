    Chip

    Passive inline label for status, tags, and active filter display. Distinct from pill (input-backed interactive). Optional close button with accessible focus ring.

    Plain Chips

    Draft In Review Approved Obsolete

    SCSS Usage

    // Default — attribute selector, no config needed
// <span data-ln-chip>Label</span>

// Tone variants
// <span data-ln-chip="success">Approved</span>
// <span data-ln-chip="warning">Pending</span>
// <span data-ln-chip="error">Rejected</span>
// <span data-ln-chip="info">Draft</span>

// With close button (focus ring auto-applied on :focus-visible)
// <span data-ln-chip>
//     Label
//     <button type="button" aria-label="Remove">
//         <svg class="ln-icon" aria-hidden="true"><use href="#ln-x"></use></svg>
//     </button>
// </span>

// Consumer semantic selector
.my-chip { @include chip; }
