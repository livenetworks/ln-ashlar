Overview & Architecture

The API Queue (data-ln-api-queue) is a connector- and id-blind offline outbox that sits between the Data Coordinator and the API Connector. It persists prepared write requests to its own IndexedDB database, replays them FIFO per record-chain with exponential backoff once connectivity returns, pauses on auth failure, and hands temp-id remapping back to the coordinator on a successful create.

Simulated Offline Mode

This playground never touches real network state. "Going offline" simply toggles data-ln-api-queue-online="false" on the queue element, which makes its internal drain guard skip network sends — exactly the override the component supports for tests.
