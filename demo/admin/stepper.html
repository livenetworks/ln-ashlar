    Stepper

    Linear progress indicator for multi-step workflows. Numbers auto-generated via CSS counter(). Three states: complete, current, upcoming.

    4-Step Approval Flow — Current: Approval

    1. Draft
    2. In Review
    3. Approval
    4. Published

    4-Step Approval Flow — Current: Draft (start)

    1. Draft
    2. In Review
    3. Approval
    4. Published

    6-Step Onboarding — Current: Connect Data

    1. Create Account
    2. Verify Email
    3. Set Password
    4. Connect Data
    5. Invite Team
    6. Go Live

    6-Step Onboarding — All Complete

    1. Create Account
    2. Verify Email
    3. Set Password
    4. Connect Data
    5. Invite Team
    6. Go Live

    SCSS Usage

    // Default attribute — zero SCSS needed
// <ol data-ln-stepper>
//     <li data-ln-step="complete"><span data-ln-step-label>Step 1</span></li>
//     <li data-ln-step="current" aria-current="step"><span data-ln-step-label>Step 2</span></li>
//     <li data-ln-step="upcoming"><span data-ln-step-label>Step 3</span></li>
// </ol>

// Consumer semantic selector
#checkout-flow { @include stepper; }

// Color override (e.g. error state on current step)
[data-ln-step="current"] { --color-primary: var(--color-error); }
