    Translations

    Inline translation system for forms. Adds translation inputs below translatable fields.

    Basics

    AttributeOn elementDescription
    data-ln-translations<form>Initializes instance
    data-ln-translations-default<form>Default language code (flag on originals)
    data-ln-translations-add<button>Trigger button (inside data-ln-dropdown)
    data-ln-translations-active<ul>Container for active language badges
    data-ln-translatable="field"wrapperMarks a translatable field
    data-ln-translations-prefixwrapperNested prefix for name attr
    data-ln-translatable-langinput/textareaLanguage code (set by JS or server)

                API & Events

                Instance API

                const el = document.querySelector('[data-ln-translations]');
el.lnTranslations.addLanguage('en');
el.lnTranslations.addLanguage('sq', { scope: 'Prodhimi...' });
el.lnTranslations.removeLanguage('en');
el.lnTranslations.hasLanguage('en');          // boolean
el.lnTranslations.getActiveLanguages();       // Set

                Events

                EventBubblesCancelableDetail
                ln-translations:before-addyesyes{ target, lang, langName }
                ln-translations:addedyesno{ target, lang, langName }
                ln-translations:before-removeyesyes{ target, lang }
                ln-translations:removedyesno{ target, lang }
                ln-translations:request-addnono{ lang }
                ln-translations:request-removenono{ lang }

                Name Generation

                // Without prefix:
name="trans[en][scope]"

// With prefix (data-ln-translations-prefix="items[5]"):
name="items[5][trans][en][title]"

                HTML Structure

                <form data-ln-translations
      data-ln-translations-default="en">
    <header>
        <h3>Section Title</h3>
        <div class="ln-translations__actions">
            <ul data-ln-translations-active></ul>
            <div data-ln-dropdown>
                <button type="button" data-ln-translations-add
                        data-ln-toggle-for="menu-id" class="ln-icon-globe"></button>
                <ul id="menu-id" data-ln-toggle></ul>
            </div>
        </div>
    </header>
    <main>
        <p data-ln-translatable="scope">
            <label>Scope <textarea name="scope">...</textarea></label>
        </p>
    </main>
</form>
