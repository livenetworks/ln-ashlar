    Overview & Philosophy

    When an action requires immediate user attention, data input, or a high-context confirmation, displaying a modal window is the standard UX pattern. However, a premium modal is more than just a centered dialog box; it requires strict accessibility and scrolling behavior controls.

    In ln-ashlar, this is managed by ln-modal. It is a highly-specialized JS component that coordinates click-triggered overlay dialogs. It handles viewport scroll locking, backdrop click dismissals, escape key tracking, and rigorous keyboard focus traps out of the box.

    The Modal Doctrine

    The contract is absolute: the active state lives solely in the data-ln-modal attribute on the modal element, and the DOM is the single source of truth. Triggers, backdrop dismissals, ESC handlers, and custom scripts all change state by writing open or closed directly to this attribute. A single MutationObserver callback runs the entire transition, scroll lock, backdrop creation, and focus management pipeline.

    Focus Trapping & Accessibility (WAI-ARIA)

    A modal must be isolated. When open, ln-modal enforces the following behavior:

    • Focus Trap: Pressing Tab or Shift+Tab cycles keyboard focus strictly through the focusable elements *inside* the modal, preventing focus from escaping into the background page.
    • Background Scroll Lock: Opening a modal immediately suppresses background scrolling by applying scroll-locking styles to the parent <body> element, preventing jarring user movements.
    • Escape Key LIFO Stack: Pressing ESC closes the most recently opened modal. If multiple stacked modals are open, it closes them sequentially (Last-In, First-Out).
    • Focus Return: When a modal closes, keyboard focus is restored precisely to the trigger element that launched it.

    For the complete developer reference, see the ln-modal README and the architecture reference.

    Code

    // Cancel opening conditionally (cancelable)
document.addEventListener('ln-modal:before-open', function(e) {
    if (e.detail.modalId === 'confirm-dialog' && !formIsValid()) {
        e.preventDefault(); // the modal will not open
    }
});

// Prevent closing (unsaved changes)
document.getElementById('edit-modal').addEventListener('ln-modal:before-close', function(e) {
    if (hasUnsavedChanges()) {
        e.preventDefault();
    }
});

document.addEventListener('ln-modal:open', function(e) {
    console.log('Opened:', e.detail.modalId);
});

document.addEventListener('ln-modal:close', function(e) {
    if (e.detail.modalId === 'confirm-dialog') {
        // reset form, clear state etc.
    }
});
