    Filter Toolbar

    Mixin-only layout pattern for list views. Arrange search, active filter chips, sort, and bulk actions into a responsive row. Consumer applies via @include filter-toolbar on a semantic selector.

    Full Toolbar — All Slots

    Status: Draft Standard: ISO 9001

    Search Only

    Search + Active Filters (No Sort/Bulk)

    Type: Procedure

    SCSS Usage

    // Apply the mixin to a semantic selector
#documents-filters { @include filter-toolbar; }

// The parent needs container-type for the 880px container query
#documents-page {
    container-type: inline-size;
}

// Resulting HTML structure
// <div id="documents-filters">
//     <div data-ln-filter-search>
//         <input type="search" placeholder="Search…">
//     </div>
//     <div data-ln-filter-group>
//         <span data-ln-chip>Status: Draft <button type="button" aria-label="Remove">…</button></span>
//     </div>
//     <div data-ln-filter-sort>
//         <select>…</select>
//     </div>
//     <div data-ln-filter-bulk>
//         <button type="button">Bulk actions</button>
//     </div>
// </div>
