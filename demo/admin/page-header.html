    Page Header

    Three-slot grid: breadcrumbs (top), title + optional subtitle (left), action buttons (right). Collapses to a vertical stack on containers narrower than 880px. Requires a container-context ancestor (container-type: inline-size).

    SCSS Usage

    // Default attribute — zero SCSS needed (requires container-type on parent)
// <div style="container-type: inline-size;">
//     <header data-ln-page-header>...</header>
// </div>

// In a project layout that already has container-type:
//   .main-content { @include container; }  ← sets container-type: inline-size
//   ↑ the page-header inside fires at 880px automatically

// Consumer semantic selector
#document-detail > header { @include page-header; }
