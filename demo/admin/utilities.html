    Utilities

    Infrastructure CSS classes that stay in HTML. For visual styling, use @include mixins in SCSS on semantic selectors.

    Infrastructure

    ClassPurpose
    .hiddendisplay: none — JS toggle target (used by ln-data-table, ln-tabs, etc.)
    .sr-onlyVisually hidden, accessible to screen readers
    .containerCentered max-width wrapper with responsive horizontal padding
    .container-smNarrow variant (56rem max-width)

    Semantic Color Overrides

    Apply to any parent element to thread --color-primary through the status color family. All child components (buttons, badges, progress bars) automatically adapt.

    ClassThreadsExample
    .success--color-success-*
    .warning--color-warning-*
    .error--color-error-*
    .info--color-info-*
    .secondary--color-secondary-*
    .neutral--color-text-muted
