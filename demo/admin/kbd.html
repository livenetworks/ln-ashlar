    Kbd

    Keycap styling for inline keyboard shortcut documentation. Applied globally to all <kbd> elements via scss/base/_typography.scss. No attribute needed.

    Basic Shortcuts

    Press Ctrl + K to open the command palette.

    Save the document with Ctrl+S.

    Undo the last action with Ctrl+Z.

    Modifier Keys

    Mac: Cmd + Shift + P

    Windows/Linux: Ctrl + Shift + P

    Open terminal: Ctrl + `

    Force quit: Cmd + Option + Esc

    Single Keys

    Navigate: Tab / Shift+Tab

    Confirm: Enter

    Cancel: Esc

    Arrow keys: ↑ ↓ ← →

    Function keys: F1 F5 F12

    Shortcuts Reference Table

    Action Shortcut
    Command palette Ctrl+K
    Save Ctrl+S
    Undo Ctrl+Z
    Redo Ctrl+Y
    Find Ctrl+F
    Bold Ctrl+B
    Italic Ctrl+I

    Kbd inside Prose

    Keyboard shortcuts

    You can use keyboard shortcuts throughout the application. Press Ctrl+K to open the command palette, or ? to see a full list of available shortcuts.

    When editing a document, Ctrl+S saves your draft, and Ctrl+Z undoes the last change.

    SCSS Usage

    // Applied globally — no attribute or class needed.
// <kbd> elements are styled automatically.

// Applied via scss/base/_typography.scss:
kbd {
  @include kbd;
}

// Re-use the mixin in consumer code:
.my-shortcut { @include kbd; }
