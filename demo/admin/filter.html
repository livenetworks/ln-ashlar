Filter

Generic filter component — filters children of a target element by data-* attribute. Works independently alongside ln-search.

Attributes

AttributeOn elementDescription
data-ln-filter="targetId"component rootTarget element by ID whose children are filtered
data-ln-filter-key="field"<button>Name of data attribute for comparison on target children
data-ln-filter-value="val"<button>Value for comparison (empty = show all)
data-ln-filter-hidetarget childrenSet by JS when element doesn't match
data-activeactive buttonSet by JS on the currently selected button

Events

document.addEventListener('ln-filter:changed', function (e) {
    console.log('Filter:', e.detail.key, '=', e.detail.value);
});

document.addEventListener('ln-filter:reset', function (e) {
    console.log('Filter reset');
});

HTML Structure

<!-- Filter buttons -->
<nav data-ln-filter="my-list">
    <button type="button" data-ln-filter-key="category" data-ln-filter-value="">All</button>
    <button type="button" data-ln-filter-key="category" data-ln-filter-value="a">Category A</button>
    <button type="button" data-ln-filter-key="category" data-ln-filter-value="b">Category B</button>
</nav>

<!-- Target list (children have data-category attribute) -->
<ul id="my-list">
    <li data-category="a">Element from category A</li>
    <li data-category="b">Element from category B</li>
</ul>

<!-- Required CSS -->
<style>
[data-ln-filter-hide] { display: none; }
[data-ln-filter-key][data-active] { background: var(--accent); }
</style>

Required CSS

The component adds attributes — the consumer provides the visual rules:

/* Hide filtered-out items */
[data-ln-filter-hide] {
    display: none;
}

/* Active filter button */
[data-ln-filter-key][data-active] {
    background: var(--color-primary);
    color: var(--color-text-on-primary);
}

/* Combined with ln-search */
[data-ln-search-hide],
[data-ln-filter-hide] {
    display: none;
}
