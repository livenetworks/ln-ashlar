    Secondary Brand Accent — --color-secondary

    The secondary color family provides a brand accent (teal) distinct from primary (blue). Apply via .secondary utility or --color-primary: var(--color-secondary) override.

    Flat Stack — flat-stack

    Cards sit flush with a single shared rule between them. Applies @include flat-stack on the parent. Self-contained: forces --radius: 0 and --shadow-resting: none on children regardless of theme.

    • Panel one

      Top child — keeps its own top border.

    • Panel two

      Top border suppressed via --border-block-start: none.

    • Panel three

      Same joined rule continues down the stack.

    SCSS Usage

    // Base card
#my-card { @include card; @include p(1rem); }

// Colored accent lines (reads --color-primary)
#alerts  { @include card; @include card-accent-top; }
#status  { @include card; @include card-accent-left; }
#tasks   { @include card; @include card-accent-bottom; }

// Override color per element
#errors  { @include card; @include card-accent-top; --color-primary: var(--color-error); }
#success { @include card; @include card-accent-left; --color-primary: var(--color-success); }

// Tinted background
#welcome { @include card; @include card-bg; }
#warning { @include card; @include card-bg; --color-primary: var(--color-warning); }

// Stacked effect
#archive { @include card; @include card-stacked; }

// Full card with header/body/footer
article {
    @include card;
    header { @include panel-header; }
    main   { @include p(1rem); }
    footer { @include px(1rem); @include py(0.75rem); @include border-t; }
}
