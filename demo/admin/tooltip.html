    Tooltip

    CSS-only tooltip via data-ln-tooltip attribute. Zero JS. Shows on :hover and :focus-visible. Four positions supported.

    Default Position — Top

    All Four Positions

    On Inline Text

    Hover over ISO 9001 to see the full standard name. Or try the semantic abbreviation pattern: ISO — an empty data-ln-tooltip falls back to the element's title.

    SCSS Usage

    // Default — attribute selector, no config needed
// <button data-ln-tooltip="Save" aria-label="Save">...</button>

// Positions
// data-ln-tooltip-position="top"    (default)
// data-ln-tooltip-position="bottom"
// data-ln-tooltip-position="left"
// data-ln-tooltip-position="right"

// Consumer semantic selector
.my-trigger { @include tooltip; }
