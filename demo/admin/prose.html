    Prose

    Rich-text content styling for TipTap/editor output. Add data-ln-prose to any wrapper containing semantic HTML.

    All Elements — Full Showcase

    Document Title (h1)

    This is the opening paragraph of a rich-text document. It uses the body-md typography role with a reading-optimized max-width of 65 characters.

    Section Heading (h2)

    Paragraphs are spaced using the > * + * lobotomised owl selector so margin is only added between sibling elements.

    A second paragraph follows. Links are styled with the primary colour and an underline. Hover thickens the underline.

    Subsection (h3)

    Content continues under the subsection heading.

    Minor heading (h4)

    The heading hierarchy maps to heading-lg, heading-md, heading-sm, and title-md typography roles.
