    acme-gui v1.2.0

    Link

    Clickable containers — the entire row/element responds to a click, finding the first <a> inside. Compatible with ln-ajax.

    Table with clickable rows

    NameEmailStatus
    Marko Petrovski marko@example.com Active
    Ana Stojanova ana@example.com Active
    Ivan Dimitrov ivan@example.com Inactive
    Elena Trajkova elena@example.com Pending

    Click anywhere on the row — it navigates to the link. Ctrl+click opens in a new tab. Hover shows the URL in the status bar.

    Rows with interactive elements

    DocumentDateActions
    Contract.pdf 15.02.2026
    Invoice.pdf 10.02.2026

    Clicking the "Delete" button triggers only the button, NOT the link. Clicking on the other cells navigates.

    Generic container

    Dashboard

    The entire container is clickable — not just the link text.

    HTML

    <!-- Table: all rows clickable -->
<table data-ln-link>
    <tbody>
        <tr>
            <td><a href="/users/1">Marko</a></td>
            <td>marko@example.com</td>
        </tr>
    </tbody>
</table>

<!-- Generic container -->
<article data-ln-link>
    <h3><a href="/post/1">Title</a></h3>
    <p>Description</p>
</article>

    Behavior

    ActionResult
    Click on rowNavigates to the first <a>
    Click on <a> directlyNormal click (not intercepted)
    Click on <button>, <input>Normal click (not intercepted)
    Ctrl/Cmd + clickOpens in a new tab
    Hover on rowURL in status bar (bottom-left)

    API

    // Manual initialization of a new DOM element
window.lnLink.init(document.getElementById('new-container'));

    Compatibility with ln-ajax

    ln-link and ln-ajax are fully decoupled — they communicate via native DOM click events:

    1. ln-link triggers .click() on <a>
    2. ln-ajax intercepts the click event (if present)
    3. If ln-ajax is not present — normal navigation
    <!-- Both components together -->
<section data-ln-ajax>
    <table data-ln-link>
        <tbody>
            <tr>
                <td><a href="/users/1">Marko</a></td>
                <td>marko@example.com</td>
            </tr>
        </tbody>
    </table>
</section>
