Confirm

A confirmation prompt for destructive actions. First click changes the button text; second click allows the action.

Basics

AttributeOn elementDescription
data-ln-confirm="Confirm?"buttonEnables confirm behavior. Value is the confirmation text.
data-confirmingbuttonAdded automatically while awaiting confirmation.

Example — Simple Buttons

Example — Inside a Form

Confirmation prevents the form from submitting on the first click.

CSS Styling

Use the [data-confirming] selector to style the button during the confirmation state.

[data-confirming] {
    background-color: var(--color-error);
    color: white;
}
