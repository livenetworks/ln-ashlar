Data Table

In-memory filter, sort, virtual scroll. Search is provided by ln-search; sort by ln-table-sort; rendering by ln-table.

Attributes

AttributeOn elementDescription
data-ln-tablewrapper (needs id)Marks the component root. Parses rows, listens for search/sort events.
data-ln-search="tableId"<input> or wrapperStandard ln-search — ln-table intercepts the event and handles filtering in-memory.
data-ln-table-clear<button>Clears the search input. Resolves via closest [data-ln-table] → id → [data-ln-search="id"].
th[data-ln-sort="string"]<th>Makes a column sortable. Values: string, number, date.
td[data-ln-value]<td>Raw sort/filter value when display text differs (formatted numbers, dates).
data-ln-table-empty<template>Content shown when search returns 0 results.

Events

EventDispatched bydetail
ln-table:readyln-table — after row parse{ total }
ln-table:filterln-table — after filter render{ term, matched, total }
ln-table:sortln-table-sort — on th click{ column, sortType, direction }
ln-table:sortedln-table — after sort render{ column, direction, matched, total }
ln-table:emptyln-table — filter returns 0{ term, total }
 
var table = document.getElementById('demo-table');

table.addEventListener('ln-table:ready', function (e) {
    countEl.textContent = e.detail.total.toLocaleString() + ' records';
});

table.addEventListener('ln-table:filter', function (e) {
    countEl.textContent = e.detail.matched < e.detail.total
        ? e.detail.matched.toLocaleString() + ' / ' + e.detail.total.toLocaleString()
        : e.detail.total.toLocaleString() + ' records';
});

data-ln-value — sort/filter raw value

When display text differs from the sort/filter key, add data-ln-value on the <td>:

CaseWithout data-ln-valueWith data-ln-value
Formatted numberSorts by "55.000" (string)Sorts by 55000 (number)
Formatted dateSorts by "15.11.2023" (string)Sorts by Unix timestamp (number)
 
<td data-ln-value="55000">55.000</td>
<td data-ln-value="1700000000">15.11.2023</td>

Virtual Scroll

Automatically activates when filtered rows exceed 200. Only visible rows + 15 buffer rows are rendered. Deactivates automatically when rows drop below the threshold.

Column widths are locked on first parse to prevent layout shifts during virtual rendering.

Dynamic Rows (AJAX)

If <tbody> is empty at init, ln-table waits — a MutationObserver detects when rows are added and parses them automatically.

// After AJAX — set innerHTML directly, ln-table will detect it
document.querySelector('#employees tbody').innerHTML = generatedHtml;
