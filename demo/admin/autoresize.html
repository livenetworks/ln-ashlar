    Autoresize

    Auto-expanding textarea — grows with content, scrolls after max height.

    Attributes

    AttributeOn elementDescription
    data-ln-autoresize<textarea>Enables auto-resize behavior — grows with content on every input event

    Example — Basic

    Example — Pre-filled

    Pre-filled content is auto-sized on initialization — no manual height needed.

    Example — Max Height with Scroll

    Pair data-ln-autoresize with CSS max-height and overflow-y: auto to cap growth and enable scroll. The <style> in this page's <head> scopes the rule to #demo-maxheight textarea.

    API

    const el = document.querySelector('[data-ln-autoresize]');
el.lnAutoresize.destroy();  // remove listener, reset height

    Behavior

    • On every input event: resets height to auto, then sets to scrollHeight
    • Pre-filled content is auto-sized on initialization
    • Works with dynamically added textareas (MutationObserver)
    • Only applies to <textarea> elements — warns if used on other tags

    HTML Structure

    <textarea data-ln-autoresize rows="1" placeholder="Type here..."></textarea>

    With max height cap (in your project SCSS or a scoped <style>):

    textarea[data-ln-autoresize] {
    resize: none;
    max-height: 6rem;
    overflow-y: auto;
}

    Dynamic Demo — MutationObserver

    Clicking the button creates a new <textarea data-ln-autoresize> via JavaScript. The MutationObserver picks it up automatically — no manual init needed.
