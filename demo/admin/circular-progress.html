    Circular Progress

    SVG-based circular progress indicator. Reactive: automatically updates when attributes change.

    Basics

    AttributeDescription
    data-ln-circular-progress="75"Current value
    data-ln-circular-progress-max="100"Maximum value (default: 100)
    data-ln-circular-progress-label="text"Custom text in center (default: percentage)

    CSS Classes

    ClassDescription
    .greenSuccess color (green)
    .yellowWarning color (yellow)
    .redError color (red)
    .smSmall size (2.5rem)
    .lgLarge size (6rem)
    .xlExtra large (8rem)

    Colors

    • Default (75%)
    • Green (90%)
    • Yellow (55%)
    • Red (25%)

    Sizes

    • .sm (2.5rem)
    • default (4rem)
    • .lg (6rem)
    • .xl (8rem)

    Custom label

    • 7 out of 10
    • Grade
    • Number

    Reactivity — click to change

    HTML Structure

    <!-- Basic usage -->
<div data-ln-circular-progress="75"></div>

<!-- With color and size -->
<div data-ln-circular-progress="90" class="green lg"></div>

<!-- With max value -->
<div data-ln-circular-progress="7"
        data-ln-circular-progress-max="10"></div>

<!-- With custom label -->
<div data-ln-circular-progress="85"
        data-ln-circular-progress-label="A+"></div>

    API

    // Manual initialization
window.lnCircularProgress(document.body);

// Change value dynamically (reactive):
var el = document.getElementById('my-circle');
el.setAttribute('data-ln-circular-progress', '80');

// Custom label:
el.setAttribute('data-ln-circular-progress-label', '8/10');

// Event:
el.addEventListener('ln-circular-progress:change', function(e) {
    console.log(e.detail.percentage); // 80
});
