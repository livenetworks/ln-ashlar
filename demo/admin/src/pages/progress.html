Progress

A visual progress indicator. Reactive: automatically updates when attributes change.

Basics

AttributeOn elementDescription
data-ln-progresstrack (empty)Marks the progress track container
data-ln-progress="50"barCurrent progress value
data-ln-progress-max="100"barMaximum value for a single bar (default: 100)
data-ln-progress-max="N"trackShared max for stacked bars — children fill proportionally

Single Progress Bars

  • Success (80%)

  • Warning (50%)

  • Error (25%)

  • Default (60%)

Custom Maximum Value

  • Storage: 7.5 GB of 10 GB

  • Backup: 45 of 50 items

Stacked Progress (Multiple Segments)

  • Resources (%) — CPU, RAM, Disk

  • Tasks — 4 done, 2 in progress, 1 pending (max=7, filled)

  • Tasks — without max (7% filled)

HTML Structure

<!-- Simple progress bar -->
<div data-ln-progress>
    <div data-ln-progress="75" class="success"></div>
</div>

<!-- Custom maximum value -->
<div data-ln-progress>
    <div data-ln-progress="7" data-ln-progress-max="10"></div>
</div>

<!-- Stacked (multiple bars) -->
<div data-ln-progress>
    <div data-ln-progress="40" class="success"></div>
    <div data-ln-progress="35" class="warning"></div>
    <div data-ln-progress="25" class="error"></div>
</div>

<!-- Stacked with shared max — fills 100% proportionally -->
<div data-ln-progress data-ln-progress-max="15">
    <div data-ln-progress="7" class="success"></div>
    <div data-ln-progress="3" class="warning"></div>
    <div data-ln-progress="5" class="error"></div>
</div>

Width Calculation

Formula: width = (value / max) * 100%

Max priority: parent track data-ln-progress-max > bar's own data-ln-progress-max > default 100.

CSS Details

Rounded corners

The last bar in a stacked progress gets a rounded right edge.

Reactivity

The component uses MutationObserver to watch data-ln-progress and data-ln-progress-max attributes on both bars and the parent track.

API

// Manual initialization
window.lnProgress(document.body);

// Dynamic update:
const bar = document.getElementById('my-bar');
bar.setAttribute('data-ln-progress', '50');
bar.setAttribute('data-ln-progress-max', '100');
