Accordion

A wrapper component that coordinates toggle elements. Only one panel can be open at a time.

Basics

AttributeOn elementDescription
data-ln-accordionparentCreates an accordion wrapper
data-ln-togglepanelChild toggle element
data-ln-toggle-forbuttonReferences the panel by ID

Example — Basic Accordion

  • What is ln-acme?

    ln-acme is a unified frontend library for LiveNetworks projects. It contains an SCSS CSS framework + vanilla JS components, with zero dependencies.

  • How to use it?

    You can use it as an npm package, via git submodule, or directly with <link> and <script> tags.

    import 'ln-acme';
import 'ln-acme/dist/ln-acme.css';
  • Does it have dependencies?

    ln-acme is fully independent. All JS components are written in vanilla JavaScript using the IIFE pattern.

    Exception: ln-select depends on TomSelect (peer dependency).

API & Events

Constructor

// Manual initialization
window.lnAccordion(document.body);

Events

Fires on the accordion container when a panel opens (after siblings are closed).

EventBubblesDetail
ln-accordion:changeyes{ target: HTMLElement }
 
document.addEventListener('ln-accordion:change', function (e) {
    console.log('Active panel:', e.detail.target.id);
});

HTML Structure

<ul data-ln-accordion>
    <li>
        <header data-ln-toggle-for="panel1">
            <strong>Title 1</strong>
        </header>
        <section id="panel1" data-ln-toggle="open" class="collapsible">
            <article class="collapsible-body">
                <p>Content 1 — starts open</p>
            </article>
        </section>
    </li>

    <li>
        <header data-ln-toggle-for="panel2">
            <strong>Title 2</strong>
        </header>
        <section id="panel2" data-ln-toggle class="collapsible">
            <article class="collapsible-body">
                <p>Content 2 — starts closed</p>
            </article>
        </section>
    </li>
</ul>

Dependencies

Accordion depends on the ln-toggle component. ln-toggle must be loaded before accordion.

<script src="dist/ln-acme.iife.js"></script>
