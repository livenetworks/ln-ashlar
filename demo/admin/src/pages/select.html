Select

TomSelect wrapper — enhances the standard <select> with search, tagging and create functionality.

Basics

AttributeOn elementDescription
data-ln-select<select>Activates TomSelect with default configuration
data-ln-select='{"create":true}'<select>JSON configuration (merged with defaults)

Default configuration

{
    "allowEmptyOption": true,
    "controlInput": null,
    "create": false,
    "highlight": true,
    "closeAfterSelect": true,
    "placeholder": "Select...",
    "loadThrottle": 300
}

Note: The placeholder attribute on <select> is used if set.

API

// Manual initialization
window.lnSelect.initialize(selectElement);

// Destroy instance (cleanup)
window.lnSelect.destroy(selectElement);

// Get TomSelect instance for direct access
var ts = window.lnSelect.getInstance(selectElement);

// TomSelect API
ts.addOption({ value: 'new', text: 'New option' });
ts.setValue('new');
ts.clear();
ts.getItems();

Graceful Degradation

If window.TomSelect does not exist, the component silently does nothing. The <select> remains functional as a regular browser select.

Best practice: Always include TomSelect before ln-acme if you are using it.

Lifecycle

On DOM remove

The instance is automatically destroyed (cleanup).

On form reset

The selection is cleared automatically.

Dynamically added select elements

New select elements with data-ln-select are processed automatically.

HTML structure

<!-- Basic TomSelect -->
<select data-ln-select>
    <option value="">Select...</option>
    <option value="option1">Option 1</option>
    <option value="option2">Option 2</option>
</select>

<!-- TomSelect with create option -->
<select data-ln-select='{"create":true}'
        placeholder="Type or select...">
    <option value="opt1">Option 1</option>
    <option value="opt2">Option 2</option>
</select>

<!-- TomSelect in a form -->
<form>
    <label for="my-select">Selection</label>
    <select id="my-select" data-ln-select name="role">
        <option value="user">User</option>
        <option value="admin">Admin</option>
    </select>
</form>

Dependencies

Peer dependency: TomSelect must be loaded before ln-acme.

<!-- TomSelect CSS -->
<link href="..." rel="stylesheet">

<script src="..."></script>
