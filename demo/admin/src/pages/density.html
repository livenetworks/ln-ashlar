Density

Global spacing compression toggle. Add the density-compact class to any element — the subtree tightens. Tables show more rows, form inputs pack denser, cards shrink padding. Same components, same markup, same selectors — only the internal spacing tokens change.

Try it

Toggle the switches below to see density in action. Global affects the whole page. Scoped affects only the Form section below — notice how the surrounding table, stat card, and intro stay comfortable while the form shrinks.

Buttons do NOT react (accessibility)

Buttons keep their full hit-target size in compact mode — WCAG 2.5.5 recommends ≥44px. Density shrinks chrome and content, never controls.
