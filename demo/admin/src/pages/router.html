Router

Overview & Philosophy

The ln-router component is a zero-dependency, **template-only** client-side router for ln-ashlar. It maps the URL to a <template data-ln-route="...">, clones it into a target element, and lets existing component MutationObservers auto-initialize the injected DOM.

In accordance with project doctrine, the router does **not** fetch data, sanitise templates, or run data-binding logic. Instead, the router mounts the view, and the **page coordinator** listens for navigation events to populate the templates using the static text interpolation helper fillTemplate.
