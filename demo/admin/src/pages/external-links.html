External Links

Utility component that automatically handles external links for security and tracking. No data attribute needed — it works automatically.

Basics

What happens?

For every link whose hostname differs from window.location.hostname:

  1. Gets target="_blank" (opens in a new tab)
  2. Gets rel="noopener noreferrer" (security)
  3. Gets marked with data-ln-external-link="processed"

Example

// Before
<a href="https://example.com">External link</a>

// After processing (automatic)
<a href="https://example.com"
   target="_blank"
   rel="noopener noreferrer"
   data-ln-external-link="processed">
    External link
</a>

// Internal links remain the same
<a href="/dashboard">Dashboard</a>

API

Manual processing of new DOM

// After dynamically adding HTML
var newContent = document.getElementById('new-content');
window.lnExternalLinks.process(newContent);

// Process the entire page
window.lnExternalLinks.process();

Dynamic content example

// Add new HTML dynamically
var html = '<a href="https://new-site.com">Link</a>';
document.getElementById('content').innerHTML += html;

// Manually process the new HTML
window.lnExternalLinks.process(
    document.getElementById('content')
);

Events

EventBubblesDetailWhen
ln-external-links:processed yes { link, href } When the link is processed
ln-external-links:clicked yes { link, href, text } When the user clicks an external link

Tracking example

// Listen when the user clicks an external link
document.addEventListener('ln-external-links:clicked', function(e) {
    var href = e.detail.href;
    var text = e.detail.text;

    // Send tracking
    console.log('External link clicked:', href, text);

    // Example: Google Analytics
    // gtag('event', 'external_link', {
    //     'event_label': text,
    //     'value': href
    // });
});

// Listen when a link is processed
document.addEventListener('ln-external-links:processed', function(e) {
    console.log('Link processed:', e.detail.href);
});

HTML example

<!-- This is all you need! The component activates automatically. -->
<nav>
    <!-- Internal link (no change) -->
    <a href="/dashboard">Dashboard</a>

    <!-- External link (will be processed) -->
    <a href="https://example.com">External Site</a>
</nav>

Security

Why rel="noopener noreferrer"?

  • noopener — prevents the new tab from accessing window.opener (XSS protection)
  • noreferrer — does not send the Referer header (privacy)

The component automatically adds these security attributes.
