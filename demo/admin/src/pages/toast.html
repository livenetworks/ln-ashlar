Toast

Notifications with a side accent and icon. They automatically disappear after 6 seconds.

Examples

To display a notification programmatically, dispatch a custom event to the window object: window.dispatchEvent(new CustomEvent('ln-toast:enqueue', { detail: { type: 'success', title: 'Title', message: 'Message' } })).

Error with a list of errors

Toasts support an array of strings in message to render a bulleted list of errors: message: ['Error 1', 'Error 2'].
