Empty State

Centered, icon-led placeholder for lists with no data. Two semantic sub-types: no-data (nothing exists yet) and no-results (filter returned zero matches).

Both Variants

No documents yet

Upload your first document to get started.

no-data — the list has never had items. Show the primary "create" action to guide the user toward the first entry.

No matches

Try a different search or clear your filters.

no-results — data exists but the current filter or search returned zero rows. Show a clear-filter action rather than a create action.
