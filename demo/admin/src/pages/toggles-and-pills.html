Native <input type="checkbox"> and <input type="radio"> elements rendered in three distinct visual treatments: Filled Pills, Outline Pills, and Switch Pills. All variants use the exact same clean, symmetric HTML markup structure and are pure CSS — no JS required.

Filled Pills (Joined Group)

Joined horizontal segmented control using ul.pills. Corner radiuses are automatically adjusted for a cohesive, joined horizontal block.

Outline Pills

Outline variant with transparent background, visible input indicator, and border highlighting. Can be grouped horizontally or used as a standalone label choice.
