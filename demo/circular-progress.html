Circular Progress

Кружен индикатор за напредок базиран на SVG. Реактивен: автоматски се ажурира кога атрибутите се менуваат.

Основи

АтрибутОпис
data-ln-circular-progress="75"Тековна вредност
data-ln-circular-progress-max="100"Максимална вредност (default: 100)
data-ln-circular-progress-label="текст"Custom текст во центар (default: проценти)

CSS класи

КласаОпис
.greenSuccess боја (зелена)
.yellowWarning боја (жолта)
.redError боја (црвена)
.smМала големина (2.5rem)
.lgГолема големина (6rem)
.xlЕкстра голема (8rem)

Бои

  • Default (75%)
  • Green (90%)
  • Yellow (55%)
  • Red (25%)

Големини

  • .sm (2.5rem)
  • default (4rem)
  • .lg (6rem)
  • .xl (8rem)

Custom label

  • 7 од 10
  • Оценка
  • Број

Реактивност — клик за промена

HTML структура

<!-- Основна употреба -->
<figure data-ln-circular-progress="75"></figure>

<!-- Со боја и големина -->
<figure data-ln-circular-progress="90" class="green lg"></figure>

<!-- Со max вредност -->
<figure data-ln-circular-progress="7"
        data-ln-circular-progress-max="10"></figure>

<!-- Со custom label -->
<figure data-ln-circular-progress="85"
        data-ln-circular-progress-label="A+"></figure>

API

// Рачна иницијализација
window.lnCircularProgress(document.body);

// Промена на вредност динамички (реактивно):
var el = document.getElementById('my-circle');
el.setAttribute('data-ln-circular-progress', '80');

// Custom label:
el.setAttribute('data-ln-circular-progress-label', '8/10');

// Event:
el.addEventListener('ln-circular-progress:change', function(e) {
    console.log(e.detail.percentage); // 80
});
